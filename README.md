## Set up
```bash
1. create a new folder  
2. open it in vs code  
3. create a new venv  
   python -m venv venv  
4. activate venv  
   venv\Scripts\Activate  
5. create the requirements.txt  
6. install packages from requirements.txt  
   pip install -r requirements.txt  
7. verify LangChain installation
```
